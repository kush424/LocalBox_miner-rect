# 💻 LocalBox Miner

## 📝 Project Description

LocalBox Miner is a React-based offline data management application designed to demonstrate:

- React State Management
- Props Communication
- CRUD Operations
- Event Handling
- Conditional Rendering
- localStorage Integration

The application allows users to:

- Add Records
- View Records in Table
- Edit Records
- Delete Individual Records
- Clear All Records
- Persist Data using Browser localStorage

⚠️ No backend or external API is used.  
All operations run completely inside React.

---

## 🧩 Project Modules

### 1️⃣ RecordForm
- Controlled Inputs
- Validation
- Add Mode / Edit Mode Switching
- onChange and onSubmit events

### 2️⃣ RecordList
- Displays records in table format
- Conditional rendering (No Records Found)
- Clear All functionality

### 3️⃣ RecordRow
- Edit Button
- Delete Button
- Event argument passing

---

## 🎯 Features Implemented

- React State Management
- Props Passing
- Event Handling (onClick, onSubmit, onChange)
- Conditional Rendering
- localStorage Read & Write
- Real-time UI Updates
- Confirmation Dialog for Clear All

---

## 📦 LocalStorage Integration

- Data is stored as JSON
- Loaded on component mount
- Updated instantly on add/edit/delete
- Data persists even after page refresh
- Data resets only when manually cleared
